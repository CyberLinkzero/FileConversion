Picking a Video Format

Use this:
  • Universal playback: MP4 (H.264 + AAC)
  • Smaller web files: WebM (VP9/AV1 + Opus)
  • Editing intermediates: MOV/ProRes
Tuning tips:
  • CRF ~23 (H.264) for balanced quality; lower = better.
  • Preset “fast/faster” for speed vs. size trade-off.
  • Use yuv420p for broad compatibility.
