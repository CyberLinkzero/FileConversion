Working with GPS Files: GPX vs KML vs TCX

Formats in a nutshell

  • GPX: Open XML for tracks/waypoints. Popular with hiking/cycling apps.
  • KML: Google Earth’s format for map features (points/paths/polygons).
  • TCX: Garmin’s training format with time/HR/cadence fields.

Convert GPS to tables

Use GPS → CSV/XLSX to flatten tracks into rows (lat, lon, elevation, time). Perfect for spreadsheets and analysis.

Convert tables back to GPS

Got a CSV with lat/lon (and optional ele/time)? Use CSV/XLSX → GPX/KML/TCX to generate clean tracks.

Tips

  • Normalize timestamps to ISO 8601.
  • Keep sampling intervals consistent to reduce jitter.
  • Validate output by loading in a map viewer before sharing.
