File Format Guide: JPG vs PNG vs WebP (& PDF basics)

Images

  • JPG: Best for photos; small size; lossy compression.
  • PNG: Best for graphics/text/screenshots; supports transparency; larger size.
  • WebP: Great balance of quality/size; supports lossy/lossless and transparency.

Need to convert? Try Images tab.

PDF essentials

Use PDF for sharing documents that must look the same everywhere. You can merge, split, or render PDFs into images for previews.

Quick tips

  • For web, prefer WebP where supported.
  • Use PNG for logos/line art; JPG for camera photos.
  • Compress PDFs before emailing if over 10MB.
