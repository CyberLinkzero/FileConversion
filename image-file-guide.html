Picking an Image Format

Use this:
  • Photos: JPG / WebP / AVIF
  • Logos & UI (needs transparency): PNG / SVG / WebP/AVIF
  • Animation: Animated WebP or APNG (GIF sparingly)
Gotchas:
  • JPEG has no transparency (white boxes behind logos).
  • PNG is lossless (crisp) but large for photos.
  • AVIF/WebP are tiny; older tools may lag.

Using the Image Converter

  1. Upload an image (JPG/PNG/WebP/AVIF/GIF/HEIC*).
  2. Pick output (PNG/JPG/WebP/AVIF). If you need alpha (transparent bg), avoid JPG.
  3. Set quality for lossy formats (start 75–85 for web).
  4. Resize or fit within a box; keep aspect ratio.
  5. For JPG, choose a background color to replace transparency.
  6. Download. Preview shows estimated size.

*HEIC support requires the built-in converter; some browsers may not decode HEIC natively.
