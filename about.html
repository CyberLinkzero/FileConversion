About File Converter — Cyber Edition

File Converter — Cyber Edition is a privacy-first, open web app for everyday conversions: images, PDFs, spreadsheets (CSV/XLSX/JSON), GPS files (GPX/KML/TCX/FIT), audio/video, and ZIP. Everything runs locally in your browser — no uploads, no accounts, no paywalls.

What makes it different?

  • 100% Local: Processing happens in-browser using Web APIs and WebAssembly (e.g., FFmpeg.wasm). Your files never leave your device.
  • Fast & Simple: Drag-and-drop and go. No watermarking, no email capture.
  • Versatile: Images ↔ formats, PDF merge/split, CSV/XLSX/JSON, GPX/KML/TCX, MP3/WAV, MP4/WebM, ZIP—more coming.
  • Open: Built for GitHub Pages; MIT-licensed where possible.

Use cases

  • Create a multi-page PDF from photos.
  • Turn a GPX hike into a clean CSV table—or export CSV back to GPX/KML/TCX.
  • Transcode a large MP4 to WebM for smaller embeds.
  • Convert CSV/XLSX to JSON for quick prototyping.

Tip: For heavy video/audio conversions, keep the tab in focus so the browser doesn’t throttle compute.
